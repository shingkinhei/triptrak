# **App Name**: TripTrak

## Core Features:

- Trip Planner: Create and manage trip itineraries with dates, locations, and activities.
- Interactive Map: Display trip locations and points of interest on an interactive map using leaflet, with custom markers. Can also be a tool to optimize your path
- Expense Tracker: Track travel expenses, categorize transactions, and generate summary reports.
- Shopping List: Create and manage shopping lists for trip essentials.

## Style Guidelines:

- Primary color: Deep orange (#E67E22), reminiscent of travel and adventure. It evokes warmth and excitement, perfect for a travel app.
- Background color: Light tan (#F5EBCD), offering a neutral backdrop that enhances readability and gives a sense of organic travel experiences (desaturated orange hue).
- Accent color: Muted gold (#D4AC0D) to highlight key UI elements, mimicking sunlight on travel destinations. The touch of yellow provides contrast while harmonizing with the overall warm scheme.
- Body and headline font: 'PT Sans', a modern sans-serif that's both clean and friendly, ideal for a travel-focused app.
- Mobile-first design with a card-based layout. The aesthetic design will mirror mobile app design best practices, will ensure easy navigation.
- Use clear, outlined icons representing travel-related items. Keep the icon style consistent and intuitive.
- Use subtle transitions and animations to enhance user experience (e.g., when opening/closing cards, displaying map markers).
